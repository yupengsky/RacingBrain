# 各向异性观测噪声

## 1. 各向异性是什么意思

各向同性噪声认为：

```text
前后左右误差一样
```

协方差像一个圆。

各向异性噪声认为：

```text
不同方向误差不同
```

协方差像一个椭圆。

本工程认为车载传感器对锥桶的纵向和横向误差不同。

## 2. 工程动机

对于赛车前方锥桶：

- 纵向距离受速度、点云稀疏、检测框深度影响更大。
- 横向位置对赛道边界更关键，通常更稳定。
- 转弯时横向漂移和姿态误差会变大。

因此代码中设计：

```text
sigma_long > sigma_lat
```

并且：

```text
sigma_lat 随 yaw rate 增大
```

## 3. 代码位置

函数：

```cpp
updateMap
```

代码：

```cpp
double sigma_long = params_.sigma_long;
double sigma_lat  = params_.sigma_lat + params_.lat_factor * std::abs(current_gyro_z_);

Eigen::Matrix2d R_body;
R_body << sigma_long * sigma_long, 0.0,
          0.0, sigma_lat * sigma_lat;
```

## 4. 参数

YAML 默认：

```yaml
sigma_long: 0.4
sigma_lat: 0.2
lat_factor: 0.5
```

含义：

- 车身前后方向标准差 0.4 m。
- 车身左右方向标准差 0.2 m。
- 转弯角速度越大，横向标准差越大。

## 5. 距离惩罚

观测距离：

```cpp
double ratio = dist_to_sensor / params_.distance_factor;
double overall_penalty = 1.0 + (ratio * ratio);
R_body *= overall_penalty;
```

距离越远，观测噪声越大。

如果：

```text
dist = distance_factor
```

则：

```text
overall_penalty = 2
```

## 6. 锁图后的速度惩罚

锁图后：

```cpp
if (map_locked_) {
    double speed_penalty = params_.speed_factor * current_speed_;
    double yaw_rate_penalty = params_.speed_factor * std::abs(current_gyro_z_);
    double dynamic_factor = 1.0 + speed_penalty + yaw_rate_penalty;
    overall_penalty *= dynamic_factor;
}
```

设计意图：

```text
冲刺圈速度更快，观测更不稳定，因此降低新观测对地图的影响。
```

## 7. 从车体系旋转到地图系

车体系噪声：

```text
R_body
```

地图系噪声：

```cpp
Eigen::Matrix2d R = vehicle_R * R_body * vehicle_R.transpose();
```

为什么？

传感器误差方向跟车头方向绑定。车辆朝向变化时，误差椭圆也应该跟着旋转。

## 8. 最小实现

```cpp
Matrix2d build_observation_covariance(
    double dist,
    double yaw_rate,
    double speed,
    bool map_locked,
    Matrix2d vehicle_R
) {
    double sigma_long = 0.4;
    double sigma_lat = 0.2 + 0.5 * abs(yaw_rate);

    Matrix2d R_body;
    R_body << sigma_long*sigma_long, 0,
              0, sigma_lat*sigma_lat;

    double ratio = dist / distance_factor;
    double penalty = 1.0 + ratio * ratio;

    if (map_locked) {
        penalty *= 1.0 + speed_factor * speed + speed_factor * abs(yaw_rate);
    }

    R_body *= penalty;

    return vehicle_R * R_body * vehicle_R.transpose();
}
```

## 9. 汇报表达

可以这样讲：

```text
传统做法可能把感知误差看成一个圆，但赛车传感器的误差其实和车身方向有关。我们在车体系下建立纵向和横向不同的误差椭圆，再根据车辆姿态旋转到地图系。这样在数据关联和滤波时，系统能更合理地区分纵向测距误差和横向边界误差。
```
