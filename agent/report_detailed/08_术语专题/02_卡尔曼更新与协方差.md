# 卡尔曼更新与协方差

## 1. 本工程使用的是简化位置 Kalman 更新

SLAM 中每个锥桶状态只有二维位置：

```text
x = [gx, gy]^T
```

每个锥桶维护一个二维协方差：

```text
P = 2x2 matrix
```

没有显式速度状态，也没有完整预测模型。每次匹配到观测后，使用 Kalman 形式进行位置融合。

## 2. 状态定义

`GlobalCone`：

```cpp
Eigen::Vector2d pos;
Eigen::Matrix2d P;
```

其中：

- `pos`：当前估计的锥桶全局位置。
- `P`：该位置估计的不确定度。

新锥桶初始化：

```cpp
new_cone.pos = z;
new_cone.P = Eigen::Matrix2d::Identity() * params_.kf_p_init;
```

## 3. 观测定义

当前帧锥桶观测：

```cpp
Eigen::Vector2d z(gx, gy);
```

观测协方差：

```cpp
Eigen::Matrix2d R;
```

`R` 来自：

- 纵向/横向基础噪声。
- 距离惩罚。
- 速度和角速度惩罚。
- 从车体系旋转到地图系。

## 4. 为什么要用协方差

协方差表达“不确定性形状”。

二维情况下：

```text
P = [ var_x    cov_xy
      cov_yx   var_y ]
```

如果：

- `var_x` 大，说明 x 方向不确定。
- `var_y` 大，说明 y 方向不确定。
- 非对角项不为 0，说明误差椭圆有旋转。

工程中观测噪声先在车体系设为对角矩阵，再旋转到地图系，因此地图系下可能出现非对角项。

## 5. 过程噪声 Q

匹配成功后先执行：

```cpp
Eigen::Matrix2d Q_noise = Eigen::Matrix2d::Identity() * params_.kf_q_base;
best_match->P += Q_noise;
```

为什么？

如果一直做 Kalman 更新，`P` 会越来越小。P 很小时，滤波器会过度相信旧地图，拒绝后续合理观测。

加入 Q 可以：

- 防止 P 过度收缩。
- 允许地图缓慢适应定位漂移或感知偏差。

## 6. Kalman 增益

代码：

```cpp
Eigen::Matrix2d S = best_match->P + R;
Eigen::Matrix2d K = best_match->P * S.inverse();
```

公式：

```text
K = P * (P + R)^-1
```

直观解释：

- 如果地图不确定度 P 大，观测噪声 R 小，K 大，更相信新观测。
- 如果地图 P 小，观测 R 大，K 小，更相信旧地图。

## 7. 位置更新

代码：

```cpp
best_match->pos = best_match->pos + K * (z - best_match->pos);
```

公式：

```text
x_new = x_old + K * (z - x_old)
```

`z - x_old` 是 innovation，也就是当前观测与预测地图位置的残差。

## 8. 协方差更新

代码：

```cpp
best_match->P = (Eigen::Matrix2d::Identity() - K) * best_match->P;
```

公式：

```text
P_new = (I - K) * P
```

更新后 P 通常变小，表示多次观测后锥桶位置更确定。

## 9. 为什么没有预测步骤

锥桶是静态地标，不像车辆有速度和运动模型。因此地图地标没有明显的状态预测：

```text
x_pred = x_old
```

工程中只通过：

```text
P += Q
```

模拟微小不确定度增长。

## 10. 与存在评分的关系

Kalman 更新解决的是：

```text
锥桶在哪里？
```

存在评分解决的是：

```text
这个锥桶是否真的存在？
```

代码中二者同时更新：

```cpp
best_match->pos = ...
best_match->P = ...
best_match->existence_score += dynamic_l_hit;
best_match->is_stable = score > l_stable;
```

所以一个锥桶要发布，需要：

- 位置被滤波。
- 多次命中达到稳定分数。

## 11. 最小实现

```cpp
void kalman_update(GlobalCone& cone, Vector2d z, Matrix2d R) {
    Matrix2d Q = Identity2 * kf_q_base;
    cone.P += Q;

    Matrix2d S = cone.P + R;
    Matrix2d K = cone.P * inverse(S);

    cone.pos = cone.pos + K * (z - cone.pos);
    cone.P = (Identity2 - K) * cone.P;
}
```

## 12. 汇报表达

可以这样讲：

```text
地图中的每个锥桶都维护二维位置和协方差。新观测到来后，如果通过马氏距离关联到已有锥桶，就用卡尔曼增益融合新旧位置。观测越可靠，更新幅度越大；地图越稳定，新观测对位置的影响越小。这样可以降低单帧感知抖动，让全局锥桶地图逐步收敛。
```
