# Agent Notes

This directory contains short reference notes for agents working on this ROS 2
workspace. Keep these files focused on project context, dependency assumptions,
and operational commands. Do not put generated build output or local machine
logs here.

Start with:

- `project_overview.md` for the package map and data flow.
- `runbook.md` for clean build and launch commands.
- `interfaces.md` for message/topic expectations and missing external inputs.
